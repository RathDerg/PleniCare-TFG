Este proyecto de carácter educativo para la asignatura de Proyecto Intermodular, la cual recopila Resultados de Aprendizaje (RA) de todas las asignaturas del ciclo formativo
de Diseño de Aplicaciones Web.

Esta aplicación web, su primera versión exclusiva para visualización en ordenadores, para una empresa de medicina privada 
llamada PleniCare (ficticia) que necesita una aplicación para que sus clientes (los pacientes) y sus operarios (los médicos) 
puedan hacer uso de una manera más interactiva y puedan pedir citas sin necesidad de llamar al teléfono, visualizarlas,
borrarlas, etc., y que a su vez, se puedan realizar los diagnósticos, los cuales podrán ser visualizados por los propios 
pacientes.

This educational project is for the Intermodular Project course, which compiles Learning Outcomes (LOs) from all the courses in the Web Application Design training cycle.


This web application, its first version exclusively for computer viewing, is for a private medical company called PleniCare 
(fictional) that needs an application so that its clients (patients) and its staff (doctors) can use it in a more interactive 
way and can request appointments without having to call, view them, delete them, etc., and in turn, diagnoses can be made, 
which can be viewed by the patients themselves.
